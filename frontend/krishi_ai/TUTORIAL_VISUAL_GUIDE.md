## Responsive Design

### Small Screens (< 360px width)
- Description box: Full width minus 32px padding
- Font sizes: Slightly reduced
- Button padding: Reduced to 16px

### Medium Screens (360px - 600px)
- Description box: Optimal size
- Font sizes: As specified
- Button padding: 20px (default)

### Large Screens (> 600px)
- Description box: Max width 400px
- Font sizes: As specified
- Button padding: 20px (default)
